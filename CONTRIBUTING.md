# Contribution Guidelines

Please submit all issues and pull requests to the [laravel/framework](http://github.com/laravel/framework) repository!

When making pull requests to this repository, please do not include the
`composer.lock` file. This is intentionally excluded so that users receive the
latest version of the framework. It is excluded from .gitignore so that users do
not have to manually add it. See #2293 for details.#2293 for details.#2293 for details.
